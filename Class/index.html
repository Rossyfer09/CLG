Lesson 5

CSS SELECTORS AND POSITIONING

CSS Selectors

We target a particular elements by class or Id, using ().) or # respectibely

Descendant Selectors

The descendant of an HTML element is any element that appears inside that element

Example

div span{ background-color: yellow}

Child Selectors

We use child selector o style Only those P elements that are direct children of the div element(only one level deep)

Example

div > P{ background-color: yellow}

We can use the same child and descendant selector patterns with CSS classes and IDs combine We can also keep listing descendant or child elements in your selectors to be more and more specific

Example

div > P span{ background-color: yellow}

Position

  • Static is the defaul Position, is in its usual Position within the page
  • RelativeDisplaces the element relative to its usual Position(Top,Left,Bottom,Right control it)
  • AbsolutePositions the element relative to its closest ancestor with a position that is not static
  • FixedKeeps the position of an elemet fixed when a user scrolls the page
